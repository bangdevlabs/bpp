################################################################
# B++ canonical font — source of truth
#
# Ancestry: Bell Labs Plan 9 terminal fonts (Lucida Typewriter
# lineage), IBM VGA 8x16 text mode (the Wolfenstein 3D era),
# LucasArts SCUMM dialogue bitmap (Monkey Island / Day of the
# Tentacle), Sierra Space Quest / King's Quest HUD fonts.
# Synthesis: 8x16 monospace, pixel-bound, programmer-friendly
# (slashed zero, spurred G, distinct i/l/1/I), slightly warmer
# than pure VGA via pixel-stepping on curves.
#
# Format:
#   Lines starting with '#' are comments.
#   Lines starting with ';;; <ascii_code>' mark a new glyph.
#   Next 16 lines are the 8x16 bitmap: '#' = ink, '.' = empty.
#   Blank lines separate glyphs (ignored by parser).
#
# Grid (8 cols x 16 rows):
#   row 0..1  = top margin
#   row 2     = top of caps / ascenders
#   row 6     = top of x-height (lowercase)
#   row 11    = baseline
#   row 12-14 = descender space (g, j, p, q, y)
#   row 15    = bottom margin
#
# To edit: tweak glyphs here, run
#   ./bpp tools/font_forge/txt_to_modulab.bpp \
#       assets/fonts/b++_font.txt \
#       assets/fonts/b++_font.modulab.json
# then open the .modulab.json in ModuLab and refine visually.
